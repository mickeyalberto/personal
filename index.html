Sistema de gestión de inventario del Hotel Real de Puebla

Este sistema fue diseñado para gestionar eficientemente el inventario del Hotel Real de Puebla, permitiendo agregar, eliminar, editar productos, generar reportes en PDF, mandar correos y mantener un control de stock automatizado.
Se utilizaron tecnologias como java 21, maven, firebase y a su vez una aplicación hecha con AndroidStudio y firebase

Sistema de punto de venta web

Este sistema fue desarrollado en función de las necesidades del cliente, buscando ofrecer un control de stock claro y ordenado, así como un servicio remoto accesible para los usuarios.
Se utilizaron tecnologías como PHP y MySQL para el almacenamiento de datos.

Aplicación balance de energia

Esta aplicación fue desarrollada para atender una de las principales necesidades de una estudiante de Ingeniería Química.
Se utilizaron tecnologías como Android Studio, Java y SQLite para el almacenamiento de datos.
